Rules

The objective of the game is to identify a 'Set' of three cards from the table. A set consists of three cards in which each property of the three cards are all the same, or all different.

Each card contains the following properties:

  1. Color: red, blue, or green.
  2. Symbol: oval, triangle, or rounded rectangle.
  3. Number: One, two, or three.
  4. Shading: Solid, open, or striped.

Scoring

  • Finding a set earns three points.
  • Getting a hint lets you know if there is a set on the table and takes away one point.
  • Asking the computer to find a set highlights a set on the table and takes away three points.
